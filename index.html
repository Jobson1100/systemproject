What is Project Engineering?

Project engineering refers to the process of managing an engineering project from start through to completion.

On a day to day basis, project engineering consists of tasks relating to: Performing calculations, Drawing up specifications,reviewing proposals etc.

While smaller projects are usually run under a single project engineer, larger projects may require a project manager
or management team.
